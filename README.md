## 🚀 快速开始

1. **克隆或下载**项目到本地
2. **确保目录结构完整**（css、js、img文件夹存在）
3. **直接用浏览器打开**任意HTML文件即可预览

> ⚠️ 注意：部分页面依赖本地图片资源（`img/`目录），如缺失图片可能显示空白，但不影响布局结构查看。

## 📚 技术栈

- **Bootstrap 4.x** - 响应式前端框架
- **jQuery 3.4.0** - JavaScript库
- **HTML5** - 语义化标记
- **CSS3** - 自定义样式增强

## 🎯 学习要点

### 响应式布局
- 使用 `col-*`、`col-sm-*`、`col-md-*`、`col-lg-*` 实现多断点适配
- 利用 `container`、`row` 构建页面骨架

### 组件应用
- **导航**：`navbar`、`nav`、`dropdown`
- **内容**：`card`、`jumbotron`、`carousel`
- **表单**：`form-control`、`input-group`、`custom-select`
- **反馈**：`modal`、`alert`、`badge`

### 实用技巧
- 自定义CSS与Bootstrap类结合使用
- 图标使用（Bootstrap Icons / SVG）
- 表单验证样式（`was-validated`）
- 进度条、表格等数据展示组件

## 🔧 自定义修改

如需调整样式，可在各文件的 `<style>` 标签内添加自定义CSS，或创建独立的CSS文件。

示例：
```css
/* 自定义主题色 */
:root {
  --primary-color: #007bff;
  --secondary-color: #6c757d;
}

/* 覆盖Bootstrap默认样式 */
.navbar {
  background-color: #000 !important;
}
